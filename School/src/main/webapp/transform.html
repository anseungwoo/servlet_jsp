transform의 변형함수

  • transform:translate(Xn,Yn); 위치 변형
    Xn,Yn 음수 가능. 단위는 px, % ,em 등
  • transform:scale(Xn,Yn); 크기 변형
    Xn,Yn은 1을 기준으로 1보다 작으면 축소, 크면 확대
  • transform:rotate(ndeg)/rotateX(ndeg)/rotateY(ndeg); 회전 변형
    n 음수 가능. 각도 단위 deg.
  • transform:skew(ndeg)/skewX(ndeg)/skewY(ndeg); 기울기 변형
    n 음수 가능. 각도 단위 deg.

transform-origin: 변형의 기준점 변경

 기본값은 정중앙(center center 또는 50% 50%)이며
 수평 수직의 순서로
 left/center/right와 top/center/bottom를 조합하거나
 npx, n%, nem, ... 등으로 값 입력해서 변경 가능
transform:translate( ) rotate( );와 transform:rotate( ) translate( );의 차이
